# Alien Symbols Language Pack

## Description

This is a joke language pack for **Minecraft** that replaces some in-game text with random alien-like symbols such as:

```
● ▲ ▼ ◆ ▰ ▮ ■
```

The goal of this pack is simply to make the game look like an **unreadable alien interface**. Menus, items, mobs, and other text may appear as strange glyphs instead of normal words.

Example:

```
Singleplayer → ●▲●
Multiplayer → ▲●▼
Diamond → ◆▰◆
Villager → ▮▲▮
```

This pack is meant for fun, experiments, screenshots, or confusing your friends. It is **not intended for practical gameplay** since most text becomes difficult or impossible to read.

---

## Disclaimer

This language pack is **not designed for real use**.
It exists purely as a **joke / novelty language** and intentionally replaces readable text with nonsense symbols.

Because of this, navigating menus or understanding messages in-game may be difficult.

Use at your own risk if you enjoy **complete UI chaos**.

---

## Alpha Status

This project is currently in **Alpha**.

* Not all Minecraft text has been translated yet.
* Some text will still appear in English.
* Missing translation keys will automatically **fall back to the default game language**.

Future updates may replace more text with alien symbols.

---

## Version Compatibility

The resource pack currently uses a **pack format intended for Minecraft 1.21+**.

However, the language file format used by this pack has existed since **Minecraft 1.13**, so the pack should still function on:

```
Minecraft Java Edition 1.13 and newer
```

Older versions may show a warning that the pack was made for a newer version, but the language translations should still work.
